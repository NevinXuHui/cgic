### **1. Windows 11 新增快捷键**

Microsoft 在 Windows 11 中添加了一些新功能。例如，Snap Layouts。 如果您将鼠标悬停在最大化按钮（每个窗口右上角关闭十字符号旁边的方块）上，您将看到多个网格。 您可以使用这些网格以您想要的方式排列您的窗口。 还有一个访问 Snap Layouts 的键盘快捷键。

|操作|快捷键|
|-|-|
|打开操作中心|Win + A|
|打开通知面板（通知中心）|Win + N|
|打开小部件面板|Win + W|
|快速访问 Snap 布局|Win + Z|
|打开 Microsoft Teams|Win + C|
