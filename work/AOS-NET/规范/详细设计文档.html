respCodenumber非必须

├─ uplinkTypestring非必须当前的上行接入方式： WLAN、Ethernet、PLC、Cable、Optical Etbernet 、GPON、XGPON、XGSPON、PoE
├─ uplinkMacstring非必须设备上行mac，格式大写去掉冒号
├─ gatewayMacstring非必须设备上行网元（网关）mac, 格式大写去掉冒号, 非PPPoE拨号时必选
├─ workingModenumber非必须AP工作模式： 0：桥接，默认 1：路由 2：中继
├─ internetTypenumber非必须AP上网方法： 0: dhcp (默认) 1: pppoe 2: static
├─ pppoeUserstring必须pppoe宽带账号,pppoe拨号是必选，非拨号时值为空

├─ radiostring非必须WLAN上联时使用的频段，取值2.4G或5G；设备工作于WLAN上联时，该字段必选；
├─ ssidstring非必须WLAN上联时接入的SSID名称；设备工作于WLAN上联时，该字段必选；
├─ securityModestring非必须认证加密模式，枚举取值： None WEP-64 WEP-128 WPA-Personal WPA2-Personal MIXED-WPAPSK2 WPA3-SAE MIXED-WPA2WPA3
├─ channelstring非必须WLAN上联时的工作信道号，如存在多信道，引用“,”逗号分隔，例：“1 ,6”；设备工作于WLAN上联时，该字段必选；
├─ noisestring非必须WLAN上联时的噪声强度，单位dBm；设备工作于WLAN上联时，该字段必选；
├─ snrstring非必须WLAN上联时的信噪比，单位dB；设备工作于WLAN上联时，该字段必选；
├─ rssistring非必须WLAN上联时的上联信号强度，单位为dBm；设备工作于WLAN上联时，该字段必选；
├─ fiberStatusnumber非必须光口连接状态，采用枚举取值： 0：Up； 1：Initializing 2：EstablishingLink 3：NoSignal 4：Error 5：Disabled
├─ fiberTXPowerstring非必须光口上联时发射光功率，单位dBm；上行光接入时必选；
├─ fiberRXPowerstring非必须光口上联时接收光功率，单位dBm；上行光接入时必选；
├─ transceiverTemperaturenumber非必须光模块的工作温度，单位：1/256摄氏度；
├─ supplyVottagenumber非必须光模块的供电电压，单位：100毫伏；
├─ biasCurrentnumber非必须光发送机的偏置电流，单位：2微安；
├─ distancenumber非必须对于光网关，该值为光网关和OLT之间的距离；单位：米； 对于关路由，该值为光路由与光网关之间的距离，单位：米；由光路由参考ITU规范的测距算法得出； 说明：该字段仅对P2MP的光路由做要求，P2P接入的光路由该值可为空；
├─ rxRatestring非必须上行协商接收速率，单位Mbps
├─ txRatestring非必须上行协商发送速率，单位Mbps
├─ rxRate_rtstring非必须上行实时接收速率(Mbps)
├─ txRate_rtstring非必须上行实时发送速率(Mbps)
├─ totalBytesSentnumber非必须上行总发送字节数
├─ totalBytesReceivednumber非必须上行总接收字节数
├─ totalPacketsSentnumber非必须上行总发送包数
├─ totalPacketsReceivednumber非必须上行总接收包数
├─ errorsSentnumber非必须上行发送错误包数
├─ errorsReceivednumber必须上行接收错误包数
├─ discardPacketsSennumber必须上行发送丢失的包数
├─ discardPacketsReceivednumber必须上行接收丢失的包数
├─ ipv6ConnStatusstring必须IPv6协议的连接状态，枚举取值： Connected Unconnected
├─ ipv6IPAddressOriginstring必须IPv6地址分配机制，枚举取值： AutoConfigured DHCPv6 Static
├─ ipv6IPAddressstring必须WAN连接的IPv6地址；
├─ ipv6DNSServersstring必须DNS Server地址。
├─ ipv6PrefixDelegationEnablednumber必须Prefix Delegation使能状态； 1:表示启用，0:表示禁用。
├─ ipv6PrefixOriginstring必须前缀地址分配机制，枚举取值： PrefixDelegation Static
├─ ipv6Prefixstring必须前缀地址。
├─ ipv6PrefixPltimenumber必须公告前缀的preferred lifetime，单位：秒 ；
├─ ipv6PrefixVltimenumber必须公告前缀的valid lifetime，单位：秒 ；
├─ defaultIPv6Gatewaystring必须IPv6默认网关 ；
├─ dsliteEnablenumber必须是否启用DS-lite功能。 1: TRUE表示启用，0: FALSE表示禁用。
